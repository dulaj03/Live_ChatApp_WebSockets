# Real time Public Chat Room using WebSockets

## Prerequisites :-
- Node.js
- Nodemon
- Sockets.io Library
- Express Framework
- html / css / js

## Install Dependancies :-
- npm i -d nodemon
- npm i socket.io
- npm i express

## Features :-
- Real time chatting
- Connecting more people
- Public chatting
- Notifications
